# Contributing to MBR WP Performance

Thank you for considering contributing!

## How to Contribute

1. **Report Bugs**: Use the [issue tracker](issues)
2. **Suggest Features**: Open a feature request
3. **Submit Pull Requests**: Fork, branch, code, PR

## Development Setup

1. Clone the repo
2. Install on local WordPress
3. Make changes
4. Test thoroughly
5. Submit PR

## Coding Standards

- Follow WordPress Coding Standards
- Comment your code
- Test on multiple WP versions
